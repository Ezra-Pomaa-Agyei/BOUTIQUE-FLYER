# Boutique Flyer
A professional, responsive boutique flyer built entirely with **Pure HTML and CSS**.

## Highlights
* **No-JS Architecture:** Focused on semantic HTML and modern CSS techniques.
* **Responsive Layout:** Designed to look great on any screen size.
* **Premium Styling:** Custom CSS for a high-end boutique look.

## Technologies Used
* HTML5
* CSS3
